# Decision-Tree


## 如何创造一棵树
检测数据集中的所有数据的分类标签是否相同:
If so return 类标签
Else:
     寻找划分数据集的最好特征（划分之后信息熵最小，也就是信息增益最大的特征）
     划分数据集
     创建分支节点
     for 每个划分的子集
          调用函数 createBranch （创建分支的函数）并增加返回结果到分支节点中
     return 分支节点
     
## 特点
优点：计算复杂度不高，输出结果易于理解，对中间值的缺失不敏感，可以处理不相关特征数据。
缺点：可能会产生过度匹配问题。
适用数据类型：数值型和标称型。
